隐私政策

Tendria 是一个本地优先、注重隐私的节奏提示工具，用于自我引导的身体觉察。

English Español 简体中文 繁體中文

概览

Tendria 不需要账号，不使用云同步，也不会加入广告或分析 SDK。应用的设计目标是让会话记录、可选备注、个人回顾、提示事件和心率样本保留在你的设备上，除非你使用的 Apple 系统功能在已配对设备之间传输数据。

Tendria 处理的信息

Tendria 可能处理会话时长、所选节奏、提示次数与提示事件、整体感受、强度、标签、可选备注、来源设备信息，以及来自 Apple Watch 的可选心率样本。

本地存储

Tendria 使用 Apple 系统存储，在你的 iPhone 上本地保存记录。只有当你选择保存记录时，备注和个人回顾才会保存。

Apple Watch 与 HealthKit

在 Apple Watch 上，Tendria 会请求读取心率数据的权限，并运行 HealthKit 体能训练会话，以便 Apple 在使用过程中提供实时心率样本。作为 Apple 体能训练会话流程的一部分，App 可能会请求共享体能训练的授权，但 Tendria 不会将体能训练记录保存到 HealthKit。心率权限是可选的，即使没有心率数据，Tendria 仍可运行会话。

Tendria 仅将心率样本用于个人回顾。Tendria 不提供诊断、治疗、医疗建议或医疗准确性声明。

从 Watch 到 iPhone 的传输

当你在 Apple Watch 上使用 Tendria 时，会话信息可能会通过 Apple 的 WatchConnectivity 系统框架本地传输到已配对的 iPhone。Tendria 不会将这些信息发送到 Tendria 服务器。

账号与云服务

Tendria 不提供账号系统，也不运营云同步服务。设备备份、iCloud 设置或“健康”App 的行为由 Apple 和你的设备设置控制，而不是由 Tendria 控制。

分析、广告与跟踪

Tendria 不加入分析、广告、跟踪像素或第三方跟踪 SDK。这些公开页面托管在 GitHub Pages。Tendria 不会在本站加入分析、Cookie 或跟踪脚本，但 GitHub 可能会根据其自身政策处理标准技术请求信息。

你主动发送的支持信息

联系支持是可选的。Tendria 只会收到你选择发送的信息。邮件会通过你的邮箱服务提供商发送，并由 Tendria 通过 Gmail/Google 接收。邮箱服务提供商可能会根据其自身政策处理和保留邮件及相关技术元数据。请不要在邮件中提供敏感的健康或私密信息、密码、付款信息或验证码。本站不包含支持表单。

数据删除

你可以在 App 中编辑或删除单条保存记录。“删除所有数据”会从 App 的本地存储中移除 Tendria 本地保存的记录，以及相关备注、个人回顾、心率样本和提示事件。

安全与设备保护

当你启用相关设置时，Tendria 可以使用设备认证来保护已保存记录。设备认证由 Apple 的 LocalAuthentication 系统处理。Tendria 不会知道或保存你的 Face ID、Touch ID 或设备密码。

儿童隐私

Tendria 不面向儿童，也不会通过 App 主动收集儿童的个人信息。如果你认为儿童在支持邮件中发送了个人信息，请通过 tendria.support@gmail.com 联系我们，以便我们妥善查看和处理相关邮件。

政策变更

随着 Tendria 的变化，本政策可能会更新。页面顶部的生效日期说明当前版本何时生效。

联系

如有隐私问题，请通过 tendria.support@gmail.com 联系 Tendria。

本政策描述 Tendria 当前版本，并非法律建议。
